⚙️ Configurações Necessárias

1. Google Maps API Key

⚠️ IMPORTANTE! Você precisa adicionar sua chave da Google Maps API:

  • Abra: contato/index.html
  • Procure por: AIzaSyDummyKeyChangeThis (linha ~84)
  • Substitua pela sua chave real
  • Veja detalhes em: README.md

2. Formspree para Formulário

Para o formulário de contato funcionar:

  • Acesse: https://formspree.io/
  • Crie uma conta e novo formulário
  • Copie seu ID do formulário
  • Abra: contato/index.html
  • Procure por: YOUR_FORM_ID (linha ~169)
  • Substitua pelo seu ID real

🚀 Como Publicar

Passo a Passo para Deploy

  • 1. Substitua Google Maps API Key
  • 2. Configure Formspree
  • 3. Ativar HTTPS (SSL) no servidor
  • 4. Ativar mod_rewrite (Apache) ou Nginx
  • 5. Fazer upload de todos os arquivos
  • 6. Testar todas as páginas em mobile
  • 7. Submeter sitemap.xml no Google Search Console
  • 8. Configurar Google Analytics
  • 9. Criar perfil Google My Business
